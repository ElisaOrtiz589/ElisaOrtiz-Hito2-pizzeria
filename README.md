# Desafío Hito 2 - Pizzería Mamma Mia: Formularios

Este proyecto corresponde al segundo hito del curso de React, enfocado en el manejo del estado y los eventos. La aplicación implementa los formularios de registro y login para una pizzería ficticia.

## Descripción del Proyecto

El objetivo principal es aplicar los conceptos fundamentales de React para crear componentes interactivos y funcionales. Se han desarrollado dos componentes principales: `Register` y `Login`, cada uno con su propio formulario y lógica de validación.

## Requerimientos Cumplidos

### 1. Formulario de Registro (`Register.jsx`)

Se ha creado un componente que renderiza un formulario de registro con las siguientes características:

*   **Campos:**
    *   Email
    *   Contraseña
    *   Confirmar contraseña

*   **Validaciones implementadas:**
    *   Todos los campos son obligatorios.
    *   La contraseña debe tener un mínimo de 6 caracteres.
    *   La contraseña y su confirmación deben ser idénticas.

*   **Comportamiento:**
    *   Al hacer clic en "Enviar", el formulario valida los datos.
    *   Si los datos son correctos, se muestra un mensaje de éxito (`Alert` color verde).
    *   Si los datos son incorrectos, se muestra un mensaje de error (`Alert` color rojo) indicando el problema.

### 2. Formulario de Login (`Login.jsx`)

Se ha creado un componente que renderiza un formulario de inicio de sesión con las siguientes características:

*   **Campos:**
    *   Email
    *   Contraseña

*   **Validaciones implementadas:**
    *   Todos los campos son obligatorios.
    *   La contraseña debe tener un mínimo de 6 caracteres.

*   **Comportamiento:**
    *   Al hacer clic en "Enviar", el formulario valida los datos.
    *   Si los datos son correctos, se muestra un mensaje de bienvenida (`Alert` color verde).
    *   Si los datos son incorrectos, se muestra un mensaje de error (`Alert` color rojo).

## Tecnologías Utilizadas

*   **React:** Para la construcción de la interfaz de usuario.
*   **React Bootstrap:** Para los componentes de UI (Form, Button, Alert).
